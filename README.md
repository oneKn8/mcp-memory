# mcp-memory

MCP server for persistent agent memory with semantic search

## Installation

```bash
pip install -e ".[dev]"
```

## Usage

```bash
mcp-memory --help
```

## Development

```bash
make install    # Install with dev dependencies
make test       # Run tests
make lint       # Run linter
make fmt        # Format code
make typecheck  # Run type checker
```
